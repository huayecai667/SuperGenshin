SuperMario
基于C++和Qt开发的经典马里奥2D平台跳跃游戏。

游戏包含完整的得分系统、生命值机制、时间限制、旗杆通关结算，以及火球发射、敌人交互等核心玩法。
在原版基础上进行了适当创新，玩家可自行切换两种不同手感的角色（Mario/Luigi）
